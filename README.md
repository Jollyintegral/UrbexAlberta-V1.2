# Other Information

**Description**
Urbex Alberta is a community-driven map designed to help explorers discover abandoned and forgotten places across Alberta, and more.

The goal is to provide a centralized, easy-to-use platform where users can:
- Find unique urbex locations
- Share discoveries with others
- Track explored vs unexplored sites

This project is built for explorers of all experience levels, with a focus on respecting locations, staying safe, and preserving history.

**Access:**
The map is publicly viewable, but certain features may require a role or access key.

Roles & Permissions:
- Viewer: Can browse locations on the map
- Member: Default when signing in. Standard permission
- Admin: Full control over the map, locations, and user access.
- Owner: All ranks, can edit code, etc. (I am the owner, the only owner and developer).

SideNote:
Roles may be assigned to maintain accuracy, prevent misuse, and protect sensitive locations.
